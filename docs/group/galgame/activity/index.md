# 活动介绍

## “十二武器”评选

<p align="center">
  <img src="/group/galgame/activity/12wp.png" style="max-width: 100%; height: auto;">
</p>

每过一段时间就会开展的galgame评选活动，名称借鉴于galgame社区中的流传的“十二神器”，详细评选规则可以进群查看哦。

“十二武器”的宗旨就是让大家畅所欲言，讨论并投票选出群内公认质量最高的十二部galgame。不过一定要有推荐的理由哦，我们还会给长评发放一些奖励（比如cd和实体galgame）。十二武器评选正是你展现厨力、舒展才华，为自己喜欢的作品拉票，撰写感想并安利其他群友入坑的大好时机！ 就算没有自己喜欢的作品上榜也不用担心，你付出心血写下的长评一样会被展示给群友，让大家了解这部作品的优秀与你对它的热爱。（别急，说不定下届就选上了呢）

## 假日歌会与聚餐

每逢佳节，群聊便会在纯k组织一次galgame主题的歌会，正是你展现厨力与歌喉的时刻（也是开盒群友的好机会）。不敢or不会唱？当然可以坐在旁边，与群友聊天吹水、指点gal业界了！顺便还能靠群友点的冷门歌曲找到一些小众佳作。嫌太远不想来？没关系，歌会后还会在学校附近举行聚餐，你可以尽情向大家推荐自己喜欢的作品，分享自己的旮旯阅历，或者拷打锐评某位你讨厌的剧本家。除此之外，群内还会在小范围不定期约k约饭，欢迎来玩！
